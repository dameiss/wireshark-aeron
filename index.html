Wireshark Dissector for the Aeron Protocol

The Aeron protocol is being developed by Real Logic. At Kaazing, we are in the process of developing a Wireshark dissector for Aeron. For the time being, the dissector is only available as a plugin module. At some point in the (hopefully near) future, the plan is to contribute the dissector code to the Wireshark project for inclusion in Wireshark itself.

As the dissector is being developed and built against the development (1.99.x) branch of Wireshark, you will need to download the latest development build for your platform.

At this time, only Linux 64-bit, Mac OSX 64-bit, and Windows 64-bit plugin modules are provided. Please don't ask for support for a different platform, or an earlier release of Wireshark.

The current dissector plugin version is 0.3.0.

Plugin version 0.3.0, released 22 April 2015

  • Reworked receiver position tracking to take into account multiple receivers (such as for multicast)
  • Reworked frame information tracking to handle the possiblity of a single network frame (packet) containing multiple Aeron frames
